Barra de herramientas

La barra de herramientas de Visual Studio Community es una interfaz que contiene varios botones y opciones rápidas para realizar tareas comunes durante el desarrollo de software. Por lo general, incluye:

  1. Archivo: Para crear, abrir, guardar o cerrar proyectos y soluciones.
  2. Edición: Herramientas para deshacer, rehacer, copiar, pegar, buscar y reemplazar.
  3. Ver: Controla las ventanas y paneles del IDE, como el Explorador de soluciones y la Terminal.
  4. Git: Opciones para gestionar el control de versiones con Git, como hacer commit, push, pull y ver el historial de cambios.
  5. Proyecto: Opciones para gestionar la configuración y las propiedades del proyecto.
  6. Compilar: Opciones para compilar y construir tu proyecto, asegurando que el código sea transformado en un ejecutable o librería.
  7. Depuración: Herramientas para iniciar, detener y depurar el código..
  8. Prueba: Herramientas para ejecutar pruebas unitarias, gestionar los resultados de las pruebas y asegurar que tu código funciona correctamente.
  9. Analizar: Opciones para analizar el rendimiento del código, el uso de memoria y otras métricas que pueden ayudar a mejorar la calidad y eficiencia.
  10. Herramientas: Configuraciones y extensiones para personalizar el entorno de desarrollo.
  11. Extensiones: Para gestionar y agregar extensiones y complementos que amplían las funcionalidades de Visual Studio.
  12. Ventana: Permite gestionar las ventanas abiertas dentro del IDE, como mover, organizar o cambiar entre ellas.
  13. Ayuda: Acceso a la documentación, recursos y soporte

Herramientas

  1. Obtener herramientas y características: Opción para añadir o quitar herramientas y características del entorno de desarrollo.
  2. Funciones de administración y revisión: Herramientas para gestionar y revisar el código, como control de versiones o análisis de calidad.
  3. Conectar con la base de datos: Opción para establecer una conexión con una base de datos para interactuar con los datos directamente desde Visual Studio.
  4. Conectar con el servidor: Herramienta para establecer conexión con un servidor, permitiendo gestionar aplicaciones y recursos remotos.
  5. SQL Server: Herramienta integrada para interactuar, administrar y depurar bases de datos SQL Server.
  6. Administrador de fragmentos de códigos: Herramienta para gestionar y reutilizar fragmentos de código predefinidos en proyectos.
  7. Elegir elementos de cuadro de herramientas: Opción para seleccionar y añadir controles y elementos a la interfaz gráfica de un proyecto.
  8. Administrador de paquetes NuGet: Opción para instalar, actualizar o eliminar paquetes NuGet en un proyecto.
  9. Crear GUID: Herramienta para generar identificadores globalmente únicos (GUID) para proyectos o recursos.
  10. Comando externo 2: Opción para ejecutar comandos externos personalizados desde el entorno de Visual Studio.
  11. Herramientas externas: Opción para integrar y gestionar herramientas de terceros dentro de Visual Studio.
  12. Línea de comandos: Herramienta para ejecutar comandos directamente desde la línea de comandos dentro del entorno de Visual Studio.
  13. Importar y exportar configuración: Opción para importar o exportar la configuración personalizada de Visual Studio.

Extensiones

Ventana

  1. Nueva ventana: Opción para abrir una nueva ventana dentro de Visual Studio para trabajar con múltiples vistas.
  2. Dividir: Herramienta para dividir la ventana activa en dos paneles, permitiendo ver diferentes áreas del código o interfaz al mismo tiempo.
  3. Hacer flotante: Opción para hacer que una ventana se desprenda del entorno principal y flote de forma independiente.
  4. Anclar pestaña: Herramienta para fijar una pestaña de ventana, asegurando que permanezca visible y accesible.
  5. Guardar diseño de ventana: Opción para guardar la disposición actual de las ventanas y pestañas para restaurarla más tarde.
  6. Aplicar diseño de ventana: Opción para aplicar un diseño de ventana previamente guardado o preconfigurado.
  7. Administrar diseños de ventana: Herramienta para gestionar y organizar los diseños de ventana guardados, permitiendo crear, modificar o eliminar diseños.
  8. Restablecer diseños de ventana: Opción para restablecer la disposición de las ventanas y pestañas al diseño predeterminado de Visual Studio.
