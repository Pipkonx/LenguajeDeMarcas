CSS

The language for styling web pages

CSS allows you to design and style your web pages.

With CSS you can transform the look of your website.

CSS is flexible and easy to learn - you'll enjoy creating beautiful designs!

CSS Property Examples

Changing the Text Color

The color property sets the color of the text of an HTML element.

Example

  1    p {

  2        color: blue;

  3    }

Changing the background color

The background-color property defines the background color of an element.

Example

  1    div {

  2        background-color: lightgray;

  3    }

Changing the font size

The font-size property specifies the size of the text in an element.

Example

  1    h1 {

  2        font-size: 36px;

  3    }

Setting margin around an element

The margin property adds outer space around an element.

Example

  1    p {

  2        margin: 20px;

  3    }

Add internal space to an element

The padding property adds space inside an element, between the content and the border.

Example

  1    div {

  2        padding: 15px;

  3    }
