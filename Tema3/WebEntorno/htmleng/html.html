What is HTML?

  • HTML significa Hiper Text Markup Language
  • HTML is the standard markup language for creating web pages
  • HTML consists of a series of elements
  • HTML elements tell the browser how to display content
  • HTML elements label pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc.

A simple HTML document

Example

  1    <!DOCTYPE html>

  2    <html>

  3    <head>

  4        <title> Page Title </title>

  5    </head>

  6    <body>

  7        <h1> My First Heading </h1>

  8        <p> My first a Paragraph </p>

  9    </body>

10    </html>

Explanation of the example

  • The <!DOCTYPE html> declaration defines that this document is an HTML5 document

  • The <html> element is the root element of an HTML page

  • The <head> element contains meta information about the HTML page

  • The <title> element specifies a title for the HTML page (which is displayed in the browser's title bar or on the page tab)

  • The <body> element defines the body of the document, and is a container for all visible content, such as headers, paragraphs, images, hyperlinks, tables, lists, etc.

  • The <h1> element defines a big heading

  • The <p> element defines a paragraph

What is an HTML Element?

An HTML element is defined by a start tag, some content, and an end tag:

The HTML element is everything from the start tag to the end tag:

Start tag Element content End tag
<h1> My first header </h1>
<p> My first paragraph </p>
<br> nothing nothing

Note: Some HTML elements have no content (such as the <br> element). These elements are called empty elements. Empty elements do not have a closing tag!

HTML Tag Examples

<table> - Create a table in HTML

The <table> tag allows you to organize data in rows and columns, creating a table structure.

Example

  1    <table>

  2    <tr>

  3      <th>Name</th>

  4      <th>Age</th>

  5    </tr>

  6    <tr>

  7      <td>Juan</td>

  8      <td>28</td>

  9    </tr>

10    </table>

Creating a form in HTML

The <form> tag allows you to create forms for users to enter data.

Example

  1    <form>

  2    <label>Name:</label>

  3    <input type="text">

  4    <button>Send</button>

  5    </form>

Including video in HTML

The <video> tag allows you to insert and control the playback of video files on the page.

example

  1    <video width="320" height="240" controls>

  2    <source src="movie.mp4" type="video/mp4">

  3    <source src="movie.ogg" type="video/ogg">

  4    Your browser does not support videos.

  5    </video>

Including audio in HTML

The <audio> tag allows you to add audio files with playback controls.

Example

  1    <audio controls>

  2    <source src="audio.mp3" type="audio/mpeg">

  3    Your browser does not support audio.

  4    </audio>

Creating a section in HTML

The <section> tag is used to group thematic content within a page.

Example

  1    <section>

  2        <h2>Title of the Section</h2>

  3        <p>This is a paragraph within the section.</p>

  4    </section>
