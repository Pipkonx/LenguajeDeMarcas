SVG

Scalable graphics for the web

SVG (Scalable Vector Graphics) allows you to create vector graphics and shapes on the web.

With SVG you can design everything from icons to complex graphics on your web pages.

SVG is ideal for graphics that need to be scaled without losing quality.

SVG Element Examples

Drawing a Rectangle

The <rect> element draws a rectangle on the screen with attributes such as width, height, and position.

Example

  1  <svg width="100" height="100">

  2    <rect width="80" height="50" fill="blue" />

  3  </svg>

Drawing a circle

The <circle> element creates a circle by defining its center and radius.

Example

  1  <svg width="100" height="100">

  2    <circle cx="50" cy="50" r="40" fill="red" />

  3  </svg>

Drawing an ellipse

The <ellipse> element draws an ellipse by setting its position, width, and height.

Example

  1  <svg width="100" height="100">

  2    <ellipse cx="50" cy="50" rx="40" ry="20" fill="green" />

  3  </svg>

Drawing a line

The <line> element draws a straight line between two points defined by their coordinates.

Example

  1  <svg width="100" height="100">

  2    <line x1="10" y1="10" x2="90" y2="90" stroke="black" stroke-width="2" />

  3  </svg>

Add text

The <text> element allows you to add text inside the SVG with position coordinates.

Example

  1  <svg width="100" height="100">

  2    <text x="10" y="50" font-family="Arial" font-size="20" fill="black">Hola SVG</text>

  3  </svg>
