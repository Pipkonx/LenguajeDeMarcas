Qué es HTML?

  • HTML significa Hiper Text Markup Language
  • HTML es el lenguaje de marcado estándar para crear páginas web
  • HTML describe la estructura de una página Web
  • HTML consiste en una serie de elementos
  • Los elementos HTML le dicen al navegador cómo mostrar el contenido
  • Elementos HTML etiqueta piezas de contenido tales como "esto es un título", "esto es un párrafo, "esto es un vínculo", etc.

Un documento HTML sencillo

Ejemplo

  1    <!DOCTYPE html>

  2    <html>

  3    <head>

  4        <title> Page Title </title>

  5    </head>

  6    <body>

  7        <h1> My First Heading </h1>

  8        <p> My first a Paragraph </p>

  9    </body>

10    </html>

Explicación del ejemplo

  • El <!DOCTYPE html> declaración define que este documento es un documento HTML5

  • El <html> elemento es el elemento raíz de un HTML página

  • El <head> elemento contiene meta información sobre la Página HTML

  • El <title> el elemento especifica un título para el Página HTML (que se muestra en la barra de título del navegador o en la pestaña de la página)

  • El <body> el elemento define la el cuerpo del documento, y es un contenedor para todos los contenidos visibles, tales como encabezados, párrafos, imágenes, hipervínculos, tablas, listas, etc.

  • El <h1> elemento define una gran partida

  • El <p> elemento define un párrafo

Qué es un Elemento HTML?

Un elemento HTML se define por una etiqueta de inicio, algo de contenido y una etiqueta final:

El elemento HTML es todo, desde la etiqueta de inicio hasta la etiqueta final:

Etiqueta de inicio Contenido del elemento Etiqueta final
<h1> Mi primer encabezado </h1>
<p> Mi primer párrafo </p>
<br> nada nada

Nota: Algunos elementos HTML no tienen contenido (como el elemento <br>). Estos elementos se denominan elementos vacíos. ¡Los elementos vacíos no tienen etiqueta de cierre!

Ejemplos de Etiquetas HTML

<table> - Crear una tabla en HTML

La etiqueta <table> permite organizar datos en filas y columnas, creando una estructura de tabla.

Ejemplo

  1    <table>

  2    <tr>

  3      <th>Nombre</th>

  4      <th>Edad</th>

  5    </tr>

  6    <tr>

  7      <td>Juan</td>

  8      <td>28</td>

  9    </tr>

10    </table>

Crear un formulario en HTML

La etiqueta <form> permite crear formularios para que los usuarios ingresen datos.

Ejemplo

  1    <form>

  2    <label>Nombre:</label>

  3    <input type="text">

  4    <button>Enviar</button>

  5    </form>

Incluir video en HTML

La etiqueta <video> permite insertar y controlar la reproducción de archivos de video en la página.

Ejemplo

  1    <video width="320" height="240" controls>

  2    <source src="movie.mp4" type="video/mp4">

  3    <source src="movie.ogg" type="video/ogg">

  4    Tu navegador no soporta videos.

  5    </video>

Incluir audio en HTML

La etiqueta <audio> permite añadir archivos de audio con controles de reproducción.

Ejemplo

  1    <audio controls>

  2    <source src="audio.mp3" type="audio/mpeg">

  3    Tu navegador no soporta audios.

  4    </audio>

Crear una sección en HTML

La etiqueta <section> se usa para agrupar contenido temático dentro de una página.

Ejemplo

  1    <section>

  2        <h2>Título de la Sección</h2>

  3        <p>Este es un párrafo dentro de la sección.</p>

  4    </section>
